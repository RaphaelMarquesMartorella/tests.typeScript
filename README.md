                                       API TESTES WITH NODE.JS, TYPESCRIPT AND JEST




                            
Clone the project

```bash
  git clone https:https://github.com/RaphaelMarquesMartorella/tests.typeSript.git
```

Go to the project directory

```bash
  cd /
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
